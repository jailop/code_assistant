# A simple code assistant

It is based on Ollama

```
Usage: code-assistant [OPTIONS] <FILENAME> <PROMPT>

Arguments:
  <FILENAME>  The name of the file to process
  <PROMPT>    The prompt to use for processing the file

Options:
      --dry-run  Enable dry run mode, no changes will be made
  -y, --yes      Automatic confirmation, skip all prompts
  -h, --help     Print help
  -V, --version  Print version
```
